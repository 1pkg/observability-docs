<div id="installation"></div>

## Install the Agent

Install the Elastic APM Go agent package using `go get`:

```bash
go get -u go.elastic.co/apm/v2
```

## Instrument Go Source Code

Instrumentation is the process of extending your application's code to report trace data to Elastic APM.
Go applications must be instrumented manually at the source code level.
There are two ways to instrument your applications:

* Using <DocBadge><DocIcon size="s" type="unlink" title="missing link"/> missing link</DocBadge>{/*  <DocLink id="enApmAgentGoBuiltinModules">Built-in instrumentation modules</DocLink> */}.
* <DocBadge><DocIcon size="s" type="unlink" title="missing link"/> missing link</DocBadge>{/*  <DocLink id="enApmAgentGoCustomInstrumentation">Custom instrumentation</DocLink> */} and <DocBadge><DocIcon size="s" type="unlink" title="missing link"/> missing link</DocBadge>{/*  <DocLink id="enApmAgentGoCustomInstrumentationPropagation">Context propagation</DocLink> */} with the Go Agent API.

Where possible, use the built-in modules to report transactions served by web and RPC frameworks in your application.
