<div id="intro"></div>

The Elastic APM Real User Monitoring (RUM) JavaScript Agent provides detailed performance metrics and error tracking of your web applications.
It has built-in support for popular platforms and frameworks, and an API for custom instrumentation.

The Agent also supports <DocBadge><DocIcon size="s" type="unlink" title="missing link"/> missing link</DocBadge>{/*  <DocLink id="enApmAgentRumJsDistributedTracingGuide">distributed tracing</DocLink> */} for all outgoing requests.
This enables you to analyze performance throughout your microservice architecture -- all in one view.

<div id="features"></div>

## Features

The agent uses browser timing APIs such as [Navigation Timing](https://w3c.github.io/navigation-timing/)
[Resource Timing](https://w3c.github.io/resource-timing/), [Paint Timing](https://w3c.github.io/paint-timing/), [User Timing](https://w3c.github.io/user-timing/), etc., and
captures the following information:

* <DocLink id="serverlessObservabilityApmAgentsRumSupportedTechnologies" section="page-load-metrics">Page load metrics</DocLink>
* Load time of Static Assets (JS, CSS, images, fonts, etc.)
* API requests (XMLHttpRequest and Fetch)
* Single page application navigations
* <DocLink id="serverlessObservabilityApmAgentsRumSupportedTechnologies" section="user-interactions">User interactions</DocLink> (click events that trigger network activity)
* <DocLink id="serverlessObservabilityApmAgentsRumSupportedTechnologies" section="user-centric-metrics">User-centric metrics</DocLink> (Long tasks, FCP, LCP, FID, etc.)
* Page information (URLs visited and referrer)
* Network connection information
* JavaScript errors
* <DocBadge><DocIcon size="s" type="unlink" title="missing link"/> missing link</DocBadge>{/*  <DocLink id="enApmAgentRumJsDistributedTracingGuide">Distributed tracing</DocLink> */}
* <DocBadge><DocIcon size="s" type="unlink" title="missing link"/> missing link</DocBadge>{/*  <DocLink id="enApmAgentRumJsBreakdownMetricsDocs">Breakdown metrics</DocLink> */}

The [APM Guide](((apm-guide-ref))/index.html) provides details on how these components work together,
