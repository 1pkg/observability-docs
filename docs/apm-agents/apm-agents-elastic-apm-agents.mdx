Elastic APM agents automatically measure the performance and track errors of your applications.
They offer built-in support for popular frameworks and technologies, and provide easy-to-use APIs that allow you to instrument any application.

## How do the agents work?

Elastic APM agents are built and maintained by Elastic. While they are similar, different programming languages have different requirements. Select your preferred language below to learn more about how the agent works.
