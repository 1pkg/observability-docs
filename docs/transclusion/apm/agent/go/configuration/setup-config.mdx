

To simplify development and testing,
the agent defaults to sending data to the Elastic APM Server at `http://localhost:8200`.
To send data to an alternative location, you must configure
<DocLink id="serverlessObservabilityApmAgentsGoConfigure" section="elastic_apm_server_url">ELASTIC_APM_SERVER_URL</DocLink>. Depending on the configuration
of your server, you may also need to set <DocLink id="serverlessObservabilityApmAgentsGoConfigure" section="elastic_apm_api_key">ELASTIC_APM_API_KEY</DocLink>,
<DocLink id="serverlessObservabilityApmAgentsGoConfigure" section="elastic_apm_secret_token">ELASTIC_APM_SECRET_TOKEN</DocLink>, and
<DocLink id="serverlessObservabilityApmAgentsGoConfigure" section="elastic_apm_verify_server_cert">ELASTIC_APM_VERIFY_SERVER_CERT</DocLink>. All other variables
have usable defaults.
