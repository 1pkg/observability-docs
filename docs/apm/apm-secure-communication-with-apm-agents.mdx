Communication between APM agents and the APM Server is both encrypted and authenticated.
Requests without a valid API key will be denied.

### Create a new API key

To create a new API key:

1. Select **Add data** -> **Collect application performance monitoring data**.
1. Under the **Configure the APM agent** step, select **Create API Key**. An API key will be generated.

Store the API key in a safe place until you're able to add it to the APM agent configuration.
The API key will not be displayed after you continue. API keys do not expire.

### Delete an API key

To delete an API key:

1. Select **Project settings** -> **Management**.
1. Under **Other**, select **API keys**.
1. Find the API key you want to delete and click on the trashcan icon.
